Shower

Full shower
Scope of Work
  • Waterproof all shower surfaces — Schluter Kerdi membrane or RedGard; flash all corners and seams
  • Build tiled bench: Wedi board or cement board substrate; waterproof; tile to match walls

  • Install 24x48 stone-look tile floor to ceiling on all three shower walls; back-butter all large format tiles

  • Install Schluter Rondec/Jolly brass edge trim at all tile perimeters and transitions
  • Apply light sanded grout on walls; dark on floor; seal all grout; caulk corners with color-matched silicone
  • Install frameless 3-panel glass enclosure with brass hardware per approved shop drawings
  • Install rain head, handheld, slide bar, and thermostatic valve; test all connections
